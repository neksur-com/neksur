# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1, as its Code of Conduct.

The full text is available at: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

Applies to all project spaces, including this repository, issue tracker, pull requests, discussions, chat channels, and any other communication channel used by the project.

## Reporting

Concerns about behavior in any project space can be reported to: `conduct@neksur.com`

Reports are handled confidentially. The maintainers will review and respond within 7 days.

## Enforcement

Maintainers will follow the Contributor Covenant 2.1 Community Impact Guidelines when determining consequences for any action they deem in violation of this Code of Conduct.
